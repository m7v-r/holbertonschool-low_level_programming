# C - File I/O

## Description
This project focuses on learning how to handle file descriptors and using system calls like `open`, `read`, `write`, and `close` in C programming under Ubuntu 20.04 LTS.

## Learning Objectives
* How to create, open, close, read and write files.
* Understanding file descriptors.
* The 3 standard file descriptors (stdin, stdout, stderr).
* Using flags: `O_RDONLY`, `O_WRONLY`, `O_RDWR`.
* File permissions and system calls vs functions.
